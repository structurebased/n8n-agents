# n8n AI Agents

Two chat agents built with n8n.

## Currency Agent
- Converts between currencies
- Uses ExchangeRate-API
- You need: API key from exchangerate-api.com

## Weather Agent  
- Shows weather for any city
- Uses WeatherAPI
- You need: API key from weatherapi.com

## Setup
1. Import JSON into n8n
2. Add your API keys in credentials
3. Add a language model (OpenRouter, Groq, etc)
4. Connect the nodes
5. Test in chat

## Issues?
These are basic templates. You'll need to connect nodes and configure your LLM.
